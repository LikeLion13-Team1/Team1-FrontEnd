
## 파일 구조
파일 구조 예시입니다.
```
TEAM1-FRONTEND/
├── assets/
│   └── salimi_icon.png       ← 이미지 등 정적 파일
│
├── js/
│   ├── index.js              ← index.html에 연결될 JS
│   └── chatbot.js            ← chatbot.html에 연결될 JS
│
├── styles/
│   ├── global.css            ← 공통 스타일
│   └── chatbot.css           ← chatbot 전용 스타일
│
├── pages/
│   └── chatbot.html          ← 챗봇 페이지
│
├── index.html                ← 메인 랜딩 페이지 (Vercel용)
└── README.md

```
