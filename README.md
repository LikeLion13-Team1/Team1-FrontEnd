
## 파일 구조
```
team1-frontend/
├── index.html
├── login.html
├── signup.html
├── mypage.html
├── chatbot.html
├── calendar.html
│
├── styles/
│   ├── global.css      ← 전체 공통 스타일
│   └── page.css        ← 각 페이지 전용 스타일 필요시 분리
│
├── js/
│   ├── common.js       ← 헤더, 네비게이션 등 공통 로직
│   └── (페이지별 js 필요시 여기에 추가)
│
├── assets/
│   └── (이미지, 아이콘 등)
```
